Cheatsheet for Web Dev

HTML Tags

Tag Name Description Example
<div></div> Div Section defines a section in a document <div class="myDiv"><p>text inside a div</p></div>
<h1></h1> Headings defines a heading <h1>my first heading</h1>
<p></p> Paragraph defines a paragraph <p>a paragraph</p>
<img> Image defines an image <img src="path/url here">
<a></a> Anchor defines a hyperlink <a href="url">link name here</a>
<br> Line Break defines a line break if i wanted a new line <br> i would use br
<video></video> Video defines embedded video content <video controls><source src="url here"></video>
<audio></audio> Audio defines embedded audio content <audio controls><source src="url here"></audio>
<-- --> Comment defines a comment <-- this is a comment -->
<button></button> Button defines a clickable button <button type="button">Click me!</button>
<span></span> Span an inline container to mark up part of a text <p> I have <span style="color:green">eyes</span></p>
